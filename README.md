## Description

Ce projet est un projet visant à récupérer des informations (Météo, news, valeurs crypto) depuis des API les transformés en un fichier txt qui sera envoyé dans une boite gmail pré défini  et une fois le script lancé cela s'execute chaque 4h apres le lancement...

---

## Installation

1. Commencez par cloner le projet avec :
    ```bash
    git clone https://github.com/Primael-dev/Qautomate.git
    ```

2.  Ouvrez votre terminal et naviguez vers le dossier du projet **`Qautomate`**.

3.  Créez un environnement virtuel pour le projet :
    ```bash
    python -m venv env
    ```

4.  Activez cet environnement :
    * **Sur Windows :**
        ```bash
        env\Scripts\activate
        ```
    * **Sur macOS ou Linux :**
        ```bash
        source env/bin/activate
        ```

5.  Installez les modules nécessaires à l'exécution du programme à l'aide du fichier `requirements.txt` :
    ```bash
    pip install -r requirements.txt
    ```

6. Créez un fichier `.env` et copier le contenu du fichier `.env.example` dans le fichier créé

7. Renseignez vos informations dans les variables.

---

## Utilisation

Une fois l'installation terminée et l'environnement activé, lancez le script en exécutant la commande suivante dans le terminal :

```bash
python app.py 
``` 
pour lancer le projet
